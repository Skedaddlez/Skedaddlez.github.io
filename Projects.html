5th Edition D&D Monster Generator (C#)

This was my first real coding project and the one that hooked me.

The code is total spaghetti, but I am proud of what I created.

The generator has the ability to produce a monster for Dungeons and Dragons based on as little information as you feed it. The idea is that the monster has a balanced stat total, based on a standard curve - this stat total is then divided as stated by the user. As such, the generated monster should be relatively safe for in-game use.

This project involved basic to moderate mathematics, programmatic file reading/writing, Windows Forms design and basic HTML.
